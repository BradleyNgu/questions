We can calculate the first couple of values of $S(n)$ to see if we can find a pattern.
$S(1) = 1$
$S(2) = 3$

<ul>
<li> $S(n)=1+logn$ 
$S(2) = 1+log{2} = 1+1 = 2$
<li> $S(n)=2+nlogn$ 
$S(2) = 2+2 log{2} = 2+2 = 4$
<li> $S(n)=2n+1$ 
$S(2) = 2 cdot 2 + 1 = 5$
<li> $S(n)=2n-1$
$S(2) = 2 cdot 2 - 1 = 3$
</ul>
Therefore, we know that $S(n)=2n-1$ is the correct answer.
